# SunaCare-Frontend

SunaCare is a mobile-first Progressive Web Application (PWA) designed to bridge the gap between injured animals and rescuers in Sri Lanka. It provides a real-time coordination platform for emergency reporting, animal welfare, and community-driven rescue efforts.

Key Features:

#One-Tap Emergency Reporting with GPS

#Real-time Interactive Rescue Map

#NGO Command Dashboard

#Verified Adoption Hub

#Transparent Donation

#Educational Resources

#Community
